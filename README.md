# Restaurant Website

A responsive restaurant website designed to showcase menu sections, services, and contact information. Built using HTML, CSS, and JavaScript with a focus on user experience.

## 🚀 Features
- Responsive layout across devices
- Menu and section-based structure
- Clean and organized design
- Basic JavaScript interactivity

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📌 Project Purpose
This project was created to strengthen my frontend development skills by building a real-world styled website from scratch.
